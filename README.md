#### ChangeLog
##### 2025.1.5 3.0.0 bate
开始升级3.0版本。
2.0最大的痛点是个人页加载慢，因为最开始的设计是后端发送全部的Log，前端来进行筛选，这个数据量很大。
先是使用 Vite SSR 开发，发现SSR是真好用啊，加载速度有明显改善。
后来接触到了Nuxt，学了一下，突然想到，Nuxt是迟早要接入的，不如早点接，现在就直接用Nuxt开始写项目。
